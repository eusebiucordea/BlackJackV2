# Blackjack Game 

A desktop application that simulates the classic Blackjack (21) card game. Built with **C#** and **.NET**, focusing on clean code architecture and Object-Oriented Programming (OOP) principles.

## Tech Stack
* **Language:** C#
* **Framework:** .NET (WinForms)
* **Design Pattern:** Object-Oriented Design (Classes for Deck, Card, Player, and Game Logic)
